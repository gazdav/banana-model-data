# Banana Model – Data

This repository contains data generated by an agent-based model
of dynamic pricing of perishable goods (bananas).

## Contents
- `BananaModelData.zip` – simulation outputs generated in MATLAB

## Usage
The data are intended for replication and further analysis
of the results presented in the associated paper.
